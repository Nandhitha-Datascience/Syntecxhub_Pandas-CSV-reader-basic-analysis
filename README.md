# 🎬 Movies Data Analysis using Python

##  Overview
This project analyzes a movies dataset using Python and Pandas.  
The goal is to explore the dataset, perform basic data analysis, and generate useful insights.

## 🛠 Tools & Technologies
- Python
- Pandas
- VS Code

##  Dataset
- moviesDB.csv  
Contains information about movies such as title, year, genres, and ratings.

##  Tasks Performed
- Loaded CSV data using Pandas  
- Inspected data using head(), tail(), and info()  
- Computed summary statistics (mean, median, min, max, count)  
- Filtered:
  - Highly rated movies (Rating > 9)
  - Movies released after 2015  
- Selected important columns (year, title, genres)  
- Created subset of first 10 rows  

##  Output Files
- highly_rated_movies.csv  
- movies_after_2015.csv  
- selected_movies.xlsx  

##  Key Insights
- Identified top-rated movies in the dataset  
- Observed trends in movies after 2015  
- Structured data for further analysis  

##  Conclusion
This project demonstrates basic data analysis skills such as data loading, cleaning, filtering, and exporting results using Python and Pandas.
